# QR-Code Generator with Python - No Expiry
## Without a logo
####  --> Go to `qrcode_without_logo.py` file.
####  --> Replace the `link` variable with your link.
####  --> Replace the `fill_color` variable with your color.
####  --> Save and run the file to get the logo saved at `main\qrcode.png`.

## With a logo
#### --> Go to `qrcode_with_logo.py` file.
####  --> Replace the `link` variable with your link.
####  --> Replace the `logo` variable with your logo.
####  --> Replace the `fill_color` variable with your color.
####  --> Save and run the file to get the logo saved at `main\qrcode.png`.
